# camel-case

Camel case a string.

## Installation

```bash
npm install camel-case --save
```

## Usage

```javascript
var camelCase = require('camel-case');

sentenceCase('string');    //=> "string"
sentenceCase('dot.case');  //=> "dotCase"
sentenceCase('PascalCase'); //=> "pascalCase"
```
